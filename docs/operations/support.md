# Support Policy

## Official support channel

Support is handled through GitHub Issues only:

- https://github.com/Zokiio/hytale-voicechat/issues

## Scope

- Bug reports for supported versions
- Operator deployment/runtime issues
- Feature requests

## Out of scope

- Private direct support
- Real-time chat support
- Security vulnerability triage in public issues

## Security reports

Use private GitHub Security Advisories for vulnerabilities:

- https://github.com/Zokiio/hytale-voicechat/security/advisories/new

See also: `SECURITY.md`.
