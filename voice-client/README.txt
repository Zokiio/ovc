Hytale Voice Chat Client
========================

Requirements:
- Java 25 or higher
- Windows, macOS, or Linux

Running the Client:
------------------

Windows:
  Double-click run.bat
  OR
  Open Command Prompt and run: java -jar HytaleVoiceChat.jar

macOS/Linux:
  Open Terminal and run: ./run.sh
  OR
  java -jar HytaleVoiceChat.jar

Configuration:
-------------
Enter your username (must match your Hytale username)
Enter the server IP address (default: localhost)
Port: 24454 (default voice server port)

Click "Connect" to join voice chat.

Troubleshooting:
---------------
If the JAR doesn't run:
1. Make sure Java 25+ is installed: java -version
2. Try running from command line: java -jar HytaleVoiceChat.jar
3. Check the server is running and the port is open
